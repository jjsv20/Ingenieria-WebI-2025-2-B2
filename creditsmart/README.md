Descripcion:
CreditSmart es un proyecto de interfaz web para visualizar, simular y solicitar créditos.Incluye un catálogo de productos financieros, un simulador de créditos y un formulario de solicitud de crédito. Todos los datos son estáticos y el proyecto está orientado unicamente a la presentación de interfaz.


Estructura:
CreditSmart/
│
├─ index.html # Página principal con catálogo de créditos
├─ simulador.html # Página del simulador de créditos
├─ solicitar.html # Página del formulario de solicitud de crédito
├─ css/
│ └─ style.css # Estilos personalizados
├─ js/ # Carpeta opcional para scripts (actualmente vacía)


Instrucciones:
1. Clonar o descargar el repositorio
2. Abrir el archivo "index.html" en el navegador de su preferencia
3. Navegar por las páginas
